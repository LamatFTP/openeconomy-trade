## SECTION 2: THE MISSION

### Section Headline
```
Why We Exist
```

### Mission Statement
```
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   "To create economic infrastructure where everyone can participate      │
│    in fair exchange, regardless of access to traditional financial       │
│    systems."                                                             │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

### The Problem We Saw
```
The traditional economy excludes millions:

• Stay-at-home parents whose labor isn't valued
• Artists and creatives between paid gigs  
• Retirees with decades of skills and no platform
• Immigrants rebuilding in new countries
• Students with time but no money
• Anyone without access to banking

Their skills have real value. 
The system just doesn't recognize it.
```

### Our Answer
```
OpenEconomy makes invisible value visible.

Instead of money, we use time-value normalization:
your skills exchanged at fair rates, person to person.

No middleman taking a cut.
No bank account required.
No barriers to entry.
```

## SECTION 4: PRINCIPLES

### Section Headline
```
What We Believe
```

### Core Principles (6)

**Principle 1: Fair by Design**
```
Icon: Scales
Description: Every system choice prioritizes fairness. 
             No features that advantage the wealthy or connected.
             Time-value normalization ensures fair exchange.
```

**Principle 2: Radically Inclusive**
```
Icon: Users
Description: If someone is excluded, we've failed. 
             No bank accounts, resumes, or credentials required.
             Design for the edges, everyone benefits.
```

**Principle 3: Transparent Always**
```
Icon: Eye
Description: Open source code. Public finances. 
             Visible reputation. No hidden algorithms.
             You can verify everything.
```

**Principle 4: Community Governed**
```
Icon: Vote
Description: The community decides the future. 
             Not founders, not investors, not corporations.
             One person, one voice.
```

**Principle 5: Privacy Respected**
```
Icon: Lock
Description: Share what you choose. 
             Your data is yours.
             We don't sell, mine, or exploit information.
```

**Principle 6: Non-Extractive**
```
Icon: Heart
Description: No fees on exchanges. No platform take rate.
             100% of value goes to participants.
             Funded by community, not extraction.
```

│ Source code        │ Proprietary     │ Open source     │
└────────────────────┴─────────────────┴─────────────────┘

### Key Differences Explained

**No Platform Fee**
```
Uber takes 25%. Fiverr takes 20%. We take 0%.

Why? Because we're not a business.
We're infrastructure, like a road or a park.
Built by the community, for the community.
```

**You Control Your Value**
```
Gig platforms race to the bottom.
Algorithms push prices down to extract more.

Here, YOU set your hourly value.
No one can tell you what you're worth.
```

**Transparent Reputation**
```
Platform ratings are often gamed or opaque.
Ours are on-chain, permanent, verifiable.
Your reputation is portable—you own it.
```
